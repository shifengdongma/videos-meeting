  5. 常见问题

  前端打不开 / 请求失败

  先确认后端是否在 127.0.0.1:8001 运行，因为前端会调用后端 API 和 WebSocket。     

  WebSocket / 直播 / 会议室异常

  直播间代码里当前连接的是：

  frontend/src/views/live/LiveRoomView.vue:144

  wsClient.connect(`ws://127.0.0.1:8001/ws/live/${liveId}`, handleSignalMessage)  

  所以后端必须先启动。

  Windows 下激活虚拟环境失败

  如果 source 不行，改用：

  . .venv/Scripts/activate
